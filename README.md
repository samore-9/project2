# Frontend Concepts Learning

This repository is my personal frontend learning journey. I'm building everything from scratch to gain hands-on experience with **HTML**, **CSS**, and **JavaScript** instead of just following tutorials.

### Goals

* Practice core frontend concepts
* Improve problem-solving through coding
* Build small projects and UI components
* Understand how websites work under the hood

### Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)

### Learning Approach

* Create each project manually
* Experiment with different layouts and designs
* Write clean, readable code
* Learn by building and fixing mistakes

This repository will continue to grow as I explore more frontend concepts and improve my development skills.
